Welcome to the Weapon Registry System!

This is an Angular application made for subject named: Webtechológiák 2

With this app, the user is able to do the following tasks:

  • User must be logged in to reach the pages
  • In the Home page the user can see, what we can do in this app
  • In the Weapon page, we can see the list of the weapons
  • In the New Weapon page , the user can add new weapons to the list ( and in the database either)
  • The user can log out, after the finished operations
  • The user can
    • Search for weapon elements
    • Modify the weapon elements
    • Delete the weapons
